## How to add a tool

1. **Fork** this repository.
2. Edit `README.md` and add your tool in the appropriate place.
3. Use the template below to keep formatting consistent.
4. Update the star/fork counts from the GitHub API or repo page.
5. Open a **Pull Request** with a short explanation of what you added.
